# TicketPlus

Java Web Service Development:Event Search and Ticket Recommendation

Designed an interactive web page for users to search events and purchase tickets utilizing AJAX technology (HTML/JavaScript); </br>
Created Java servlets with RESTful APIs to handle HTTP requests and responses;</br>
Built relational and NoSQL databases (MySQL, MongoDB) to capture event data from TicketMaster API;</br>
Designed algorithms (e.g., content-based recommendation) to implement event recommendation;</br>
Deployed server to Amazon EC2 to handle 150 queries per second tested by Apache JMeter.</br>
